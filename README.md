# Different-mini-games

OVERVIEW OF THE PROJECT
Different mini Games contains (NUMBER GUESSING),(DICE ROLLING) ,(WATER SNAKE GUN ) is a Python-based interactive game designed for computer and  player. The game selects a random word from a predefined list, jumbles the letters, and challenges the players to guess the correct word. The player who correctly identifies the word get  a point. The game continues until the players choose to quit.

FEATURE 
Player and computer  interact in the game.
 Provide option for the player which game he or  she likes to play
Random word selection from a list of words.
Shuffling/jumbling of selected word letters.
Turn-based gameplay
Winner declaration at the end of the game.
Option to continue or quit after each round

TECHNOLOGIES USED
	Programming Language: Python 3.x
	Libraries Used: random (for selecting and shuffling words)
	IDE/Environment: Any Python IDE or text editor (e.g. Google Colab)
